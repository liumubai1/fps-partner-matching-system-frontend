## 项目特点
<font style="color:rgb(31, 35, 40);">该项目基本覆盖企业开发的常见需求 以及对应的解决方案，比如用户登录、批量数据导入、信息检索展示、定时任务、资源抢占等。并且涵盖了分布式、并发编程、锁、事务、缓存、性能优化、幂等性、数据一致性、大数据、算法等的知识与实践。</font>
